# A simple p5.js portfolio

Developed at ITP Summer Camp 2018

## Requirements

Run this in you command line

1. `brew install git`
2. `curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.11/install.sh | bash`
3. `nvm install node`

## Usage

1. `git clone `
2. `npm install`
3. `npm start`
